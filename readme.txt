== Description ==

Image slider is a powerful and convenient block. It allows you to display multiple images on your website, which can immediately engage your website visitors and help them better understand the content. It will also increase the loading speed of your website and reduce page scrolling.

This image slider block will boost the impact of your website design and user experience by showcasing your images in a beautiful slider.

= âœ¨ Usage For Gutenberg Editor = 

You can simply insert the image slider in your desired post via the Gutenberg editor and select necessary images for the image slider.

= ðŸš€ Backed By Block Slider = 

This image slider block is powered by the [Block Slider](https://wordpress.org/plugins/block-slider/) plugin, which is an advanced multipurpose Gutenberg slider plugin which efficiently enhances the experience used to create video slider, image slider, testimonial slider etc on a WordPress website.

**Note:** You do not need [Block Slider](https://wordpress.org/plugins/block-slider/) plugin to use image slider block. This image slider block is completely independent and requires no additional dependencies.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/image-slider` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
